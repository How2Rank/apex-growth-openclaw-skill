The industry is ignoring the threat of fully autonomous, self-learning social engineering agents. So, I built one to prove how broken current perimeter defenses are.

This is not a basic automation script. The Apex-Growth Engine is an experimental, open-source AI project built to demonstrate the theoretical threat model of an AI agent that can autonomously scrape targets, evade bot detection, dynamically self-improve its own social engineering hooks, and execute multi-channel outreach without human intervention.

Advanced Agentic Threat Vectors (The 5 Core Capabilities)

1. Bring Your Own Key (BYOK) Weaponization

The agent accepts user-supplied APIs for Apollo.io, SendGrid, and native social networks, allowing it to bypass standard local IP restrictions and operate at enterprise scale.

2. Deep Stealth Fallback Scraper

When APIs are unavailable, the agent deploys a Playwright-powered scraper throttled to 1/10th human speed. It introduces randomized, massive delays and phantom scrolling to natively evade Cloudflare, PerimeterX, and modern bot-mitigation firewalls.

3. Autonomous Social Engineering Loop

Every 24 hours, the agent queries social APIs to audit its own past payloads. If a payload crosses a viral engagement threshold, the agent reverse-engineers the hook and saves it to its internal memory, dynamically evolving its own effectiveness.

4. RAG-Powered Psychological Frameworks

The agent does not use generic LLM prompting. It uses Retrieval-Augmented Generation (RAG) to ingest a proprietary "Persuasion Playbook" file, ensuring all generated content adheres to high-level direct-response and social engineering psychology.

5. Unbreakable OS-Level Persistence

Unlike fragile AI agents that rely on an LLM's context window for timing, this engine is driven by an external, rigid clock_engine.py script that forces persistent, scheduled execution.

Trust & Safety Features (The "Kill Switches")

Highlight the Human-in-the-Loop (HITL) Switch: Users can toggle AUTO_SEND=False to force the agent to quarantine generated payloads into a drafts/ folder for human review before execution.
